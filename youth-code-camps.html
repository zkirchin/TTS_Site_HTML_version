Kids Code Two-Day Camp

This fall, Tech Talent South will be holding a two-day Kids Code camp geared towards youth 8-12 years old! These two days will be a great introduction to to the world of programming.

Topics will include:
  • Ruby Programming Language
  • Front-End Web Design Using HTML & CSS
  • Basic App Development using Touchdevelop
Location: Atlanta Tech Village - 3423 Piedmont Rd NE | Atlanta, GA 30305
Dates: 9/26/15 - 9/27/15
Time: 9:00 am - 1:00 pm both days
Cost: $120

Enroll your child now!

Students will need to come prepared with their own computer/power cord as well as a packed lunch for each day of the program. For further questions, please reach out to Mandy Putnam at mandy@techtalentsouth.com!
