What will I learn? Well...a lot!

Ruby on Rails

Get on board with this insanely popular and practical open source web application framework that has been used to build tools like Twitter, Basecamp, Hulu, Groupon, and Shopify.

Domain Modeling

Learn the crucial skill of modeling resources and relationships in a dynamic, database-backed web application.

HTTP and "RESTful" Design

Understand how the crazy world of the Internet works and how superior web applications are designed.

HTML/CSS

Learn how these languages are used to paint beautiful websites and user experiences.

Javascript and AJAX

Use languages and techniques to make your web application interactive and seamless.

Understanding and Utilizing APIs

Learn how to integrate your web application with tools like Facebook and Google Maps.
